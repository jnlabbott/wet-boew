Overview

The Notification Popup currently uses Toastr to provide a popup notification mechanism.

Examples

Success Notification Alert Notification Error Notification Info Notification Success Notification with HTML

Usage

The notification plugin must first be initialized by called the pe.notify_init() method on or after document ready.

For example:

$(function() {
  pe.notify_init();
});

To display a popup in response to an event call the pe.notify(sType, STitle, sMessage) function with 'sTypes' being a string type of notification.

Currently supported types are: Success, Alert, Info and Error.

pe.notify('success', 'Woohoo!', 'Great, that was a success!');

For example, say you wanted to display a success notification popup after a click event that updates some information through AJAX:

<a href="#" class="button" onclick="update()">Save Changes</a>
<script type="text/javascript">
  function update()
  {
    var updateCompleted = false;
    // do some update activity
 
    if updateCompleted
      pe.notify('success', '', 'Update Completed.');
   }
</script>
